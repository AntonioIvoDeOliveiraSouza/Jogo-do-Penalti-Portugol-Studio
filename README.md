# Jogo de Pênaltis em Portugol Studio
Este é um jogo de pênaltis desenvolvido durante o primeiro módulo do IFPI campus Parnaíba. O jogo é inspirado no esporte de futebol e foi criado em pseudocódigo (Portugol Studio).

# Descrição
O jogo simula uma disputa de pênaltis entre o jogador e a máquina (goleiro). O objetivo é marcar o máximo de gols possível ao chutar a bola para um dos cantos do gol, enquanto o goleiro tenta defender.

# Funcionalidades
Escolha entre ser o batedor ou o goleiro.
Chute a bola para um dos cantos do gol.
Goleiro tenta defender o chute adversário.
Os acertos e erros são registrados em tabelas.
Modo de pênaltis alternados em caso de empate.

# Instruções de Uso
Execute o programa no Portugol Studio.
O menu será exibido, escolha a opção "INICIAR" (opção 1) para começar o jogo.
Siga as instruções na tela para bater ou defender pênaltis.
Os resultados dos chutes e defesas serão exibidos durante o jogo.
Após o término das rodadas, o vencedor será anunciado.

# Como Jogar
Como Batedor (Jogador)

Escolha um dos cantos do gol para chutar (números de 1 a 5).
Se o goleiro defender ou se você errar o chute, um símbolo "X" será exibido na tabela de acertos/erros.
Se você marcar um gol, um símbolo "⊛" será exibido na tabela de acertos/erros.

Como Goleiro

Escolha um dos cantos do gol para defender (números de 1 a 5).
Se você defender o chute do adversário, um símbolo "X" será exibido na tabela de acertos/erros.
Se o adversário marcar um gol, um símbolo "⊛" será exibido na tabela de acertos/erros.

# Resultados
Ao final das rodadas, os acertos e erros de ambos os jogadores serão exibidos.
O jogador com mais gols marcados é declarado vencedor.
Em caso de empate, o jogo entrará no modo de pênaltis alternados.

# Requisitos
Portugol Studio (ou qualquer ambiente compatível com pseudocódigo).
